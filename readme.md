# Víno Kostor
Webová appka, ktorá pokrýva základné funkcionality pre potreby evidendovania dát okolo nášho vínka.

Skladá sa z prezentačnej a administračnej časti.

## Tehcstack
Appka je napísaná v PHP 8 a Symfony 5. Používa MySQL databázu.

Na FE je použitý Tailwind CSS a Alpine.js doplnený o jQuery.

## Nasadenie
V optimistickej budúcnosti bude appka zdockerizovaná a nasadená na adrese [vinokostor.sk](http://vinokostor.sk). Aktuálne je na tej adrese dostupná stará verzia našej stránky...
